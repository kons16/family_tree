# family_tree 
`family_tree.py` は家系図を作成できるGUIアプリです.

## 使用方法
1. メインの人物名を入力し, `メイン人物の追加`をクリック.
1. 名前を入力し, `追加`をクリック.
1. キャンバス上で人物をクリックし,`4`で入力した人物の位置を選ぶ.
1. 以降`2`,`3`を繰り返す.
